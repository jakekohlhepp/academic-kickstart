# Notes workflow

Use this folder as the intake area for your own free-flow thoughts.

## Default flow

1. Keep writing in your Google Doc however you want.
2. When you want the wiki updated, paste or export the latest text into `notes/inbox/google-doc-notes.md`.
3. Tell me to integrate the notes.

## What I will do during integration

- pull out useful claims, questions, and connections
- place article-specific notes into relevant article pages
- add cross-paper ideas to theme pages
- keep a raw archive copy when useful

## Honest limitation

There is not a live Google Docs API connection wired into this workspace right now. The current setup is built for a very low-friction handoff instead: a pasted export or a public/published Google Doc link.
